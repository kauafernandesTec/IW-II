Introdução ao DOM (JavaScript)

Objetivo:

Compreender o que é o DOM e como o JavaScript interage com elementos HTML.

Pesquisa:

1. O que é DOM? Explique com suas palavras e a relação com HTML.

O DOM é a forma como o navegador organiza o HTML em uma árvore de objetos. Com isso, o JavaScript consegue acessar e modificar elementos da página (texto, estilo, estrutura).

2. O que é um nó (node)? Explique elementos, atributos e textos.

Um nó é qualquer parte do DOM. Os principais são: Elementos → as tags HTML (div, p) Atributos → informações das tags (id, class) Texto → conteúdo dentro das tags

3. Explique os métodos: getElementById(), querySelector(), querySelectorAll().

getElementById() → seleciona um elemento pelo id (retorna só um) querySelector() → retorna o primeiro elemento que corresponde ao seletor CSS querySelectorAll() → retorna todos os elementos (em forma de lista)

4. Diferença entre innerHTML e innerText.

innerHTML → lê ou altera o conteúdo incluindo tags HTML innerText → lê ou altera apenas o texto visível, sem interpretar HTML

5. O que são eventos? Exemplos: onclick, onchange, onmouseover.

Eventos são ações que acontecem na página e podem executar código JavaScript: onclick → quando clica onchange → quando um valor muda onmouseover → quando o mouse passa por cima
